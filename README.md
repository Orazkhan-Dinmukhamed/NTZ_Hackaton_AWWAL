SOZTYPE

Комплексная платформа для работы с научными проектами и техническими заданиями (ТЗ), включающая AI-проверку заявок и AI-редактор документов.

---

# 📁 Структура проекта


# 🚀 О проекте

Система состоит из двух основных частей:

## 1️⃣ SozType App (AI Editor)

📂 `/soztype-app`

👉 Умный редактор научных документов (аналог Google Docs)

### Возможности:
- ✍️ Написание научных ТЗ
- 🧠 AI-ассистент (Groq / LLM)
- 📋 Генерация разделов:
  - Цель
  - Задачи
  - Методология
  - Результаты
- 📄 Экспорт в PDF (A4 формат)
- ⚡ Быстрые кнопки генерации текста
- 🎯 Полная структура научного документа

---

## 2️⃣ Application System (AI Review)

📂 (часть внутри проекта / серверная логика)

👉 Система подачи и проверки научных заявок

### Возможности:
- 📤 Загрузка PDF / DOCX
- 🔍 Парсинг текста
- 🧠 AI-анализ соответствия ТЗ
- ✅ / ❌ Автоматическое решение (Approved / Rejected)
- 💾 Сохранение в Supabase
- 📊 Комментарии и причины отклонения

---

# 🧠 Как работает система

### Пользовательский поток:

1. Пользователь пишет или загружает научный проект
2. AI проверяет структуру ТЗ
3. Система определяет:
   - подходит ли проект
   - ошибки и несоответствия
4. Результат:
   - ✅ Approved
   - ❌ Rejected (с причиной)

# ⚙️ Технологии

- HTML / CSS / JavaScript
- Vercel (serverless API)
- Supabase (database + storage)
- Groq API (LLM AI)
- Gemini API (validation logic)
- PDF export engine (A4 format)

---

# 🗄️ База данных

Используется единая Supabase база:

- заявки пользователей
- загруженные файлы (PDF / DOCX)
- распарсенный текст
- AI анализ
- статус проекта

---

# 📄 PDF система

- строгий формат A4
- белый фон
- без серых полей
- печатный вид как официальный документ

Создать интеллектуальную систему, которая:

- ускоряет написание научных ТЗ
