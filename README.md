# ✨ SozType

**SozType** — это система из трёх связанных приложений для подачи, анализа и редактирования научных ТЗ и работ.

---

## 🔗 Демо

* [SozType Edit (index1)](https://soztype-edit.vercel.app/index1.html)
* [SozType User](https://soztype-edit.vercel.app/)
* [SozType Admin](https://soztype-app.vercel.app/)

---

## 📌 О проекте

SozType создан для удобной работы с научными проектами и техническими заданиями.

Система состоит из трёх частей:

1. **SozType** — приложение для пользователя.
2. **SozType Admin** — приложение для админов и проверки заявок.
3. **SozType EDIT** — редактор документов, заточенный под научные ТЗ.

---

## 👤 1) SozType — приложение для пользователя

Пользовательская часть предназначена для регистрации, подачи заявок и отслеживания статуса своих работ.

### Основные функции 🧩

* Регистрация и вход в аккаунт.
* Push-уведомления с последними событиями.
* Раздел **Мой аккаунт** для настройки уведомлений и выхода из системы.
* Главная статистика по заявкам:

  * всего отправлено;
  * на рассмотрении;
  * принято.
* История последних отправленных работ с результатами:

  * принято;
  * отказано;
  * на рассмотрении.
* Форма **Подать заявку** с полями:

  * контакты;
  * ФИО;
  * ИИН;
  * название работы;
  * описание;
  * файл работы в формате PDF или Word.
* После отправки заявка уходит в систему анализа.
* Пользователь сразу видит текущий статус заявки: **на рассмотрении**, **принято** или **отказано**.

---

## 🛠️ 2) SozType Admin — приложение для отсеивания работ

Админ-панель предназначена для быстрой сортировки и анализа поступающих работ.

### Основные функции

* Понятный интерфейс со списком всех загруженных работ.
* Быстрый просмотр статусов:

  * принято;
  * отказано;
  * на рассмотрении.
* Поиск по заявкам.
* Фильтрация по статусам.
* Сводная статистика сверху:

  * всего работ;
  * принято;
  * отказано.
* При открытии работы отображаются:

  * все данные пользователя;
  * время отправки;
  * описание;
  * прикреплённый файл;
  * контакты;
  * ИИН.
* Возможность сразу скачать прикреплённый документ.
* Встроенный AI-анализ:

  * автоматически парсит документ;
  * оценивает по критериям ТЗ научного проекта;
  * принимает решение за несколько секунд;
  * выдаёт балл по 100-балльной шкале;
  * формирует краткий конспект;
  * объясняет причины отказа или принятия.

### Пример AI-анализа

* Общая оценка: **0 / 100**
* Решение: **Отклонён**
* Причина: документ является пустым шаблоном без заполненных данных.
* Дополнительно AI показывает:

  * ошибки;
  * предупреждения;
  * оценку по критериям;
  * краткое описание проекта.

### Логика AI

AI работает в строгом режиме и оценивает документ именно как **научное ТЗ**. Он не просто выносит решение, а помогает понять, что именно не так в работе.

## 📝 3) SozType EDIT — редактор для пользователей

SozType EDIT — это аналог Google Docs, но адаптированный под научные ТЗ и документы.

### Основные функции

* Редактирование текста в удобном документ-редакторе.
* Почти полный базовый функционал Google Docs.
* Готовая кнопка **шаблон ТЗ** для быстрой вставки нужной структуры.
* AI-помощник справа:

  * отвечает на вопросы по документу;
  * помогает писать ТЗ;
  * предлагает шаблоны;
  * делает умный поиск;
  * подсказывает, как правильно оформить документ.
* Возможность переименовывать документ.
* Экспорт и скачивание в PDF.

---

## 🔄 Как это работает вместе

1. Пользователь регистрируется или входит в аккаунт.
2. Заполняет заявку и прикрепляет файл.
3. Заявка отправляется в систему.
4. Админ получает её в **SozType Admin**.
5. AI анализирует документ и выносит решение.
6. Пользователь видит статус и результат проверки.
7. При необходимости пользователь дорабатывает документ в **SozType EDIT**.

## ⚙️ Установка и запуск

### 1. Подключите Supabase ☁️

Нужно создать и подключить собственный проект Supabase.

### 2. Добавьте SQL 🧾

Используйте SQL-код, который вы уже подготовили для базы данных.

### 3. Настройте переменные окружения 🔐

Для деплоя через Vercel добавьте переменные в `.env`:

```env
groq_api=YOUR_GROQ_API_KEY
SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
SUPABASE_URL=YOUR_SUPABASE_URL
```

### 4. Деплой 🚀

* Деплойте через **Vercel**.
* Для **SozType EDIT** подключите ключ `groq_api`.
* Для **SozType** и **SozType Admin** используйте Supabase-ключи и URL.

---

## 🔑 Переменные окружения

| Переменная          | Назначение                  |
| ------------------- | --------------------------- |
| `groq_api`          | Ключ для AI в SozType EDIT  |
| `SUPABASE_ANON_KEY` | Публичный ключ Supabase     |
| `SUPABASE_URL`      | URL вашего проекта Supabase |
